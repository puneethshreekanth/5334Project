How to run:
1. Run the run.py file using "python run.py"
2. The following output will displayed.
	* Running on http://0.0.0.0:5000/ (Press CTRL+C to quit)
 	* Restarting with stat
 3. Open http://0.0.0.0:5000/ in the browser of your choice.
 4. The index.html page will load.
 5. Choose the user
 6. suggestions.html will provide a list of spots, where the user can next go, based on his/her previous visits.
